# ⚡ Your Socials OS

## Setup — ONE command

```bash
node setup.js
```

That's it. The script will:
1. ✅ Check Git + Node are installed
2. ✅ Ask your name, email, and password (2 minutes)
3. ✅ Generate all secrets automatically
4. ✅ Deploy to Vercel
5. ✅ Set all environment variables
6. ✅ Create your admin account
7. ✅ Open the app in your browser

---

## Requirements before running

- [Node.js](https://nodejs.org) installed (v18+)
- [Git](https://git-scm.com) installed
- A [GitHub](https://github.com) account (free)
- A [Vercel](https://vercel.com) account (free)

---

## After setup

Log in → go to **⚙️ Manage Users** in the sidebar → add your team.

Each team member gets:
- A name, email, and password (you set these)
- Role: Admin / Member / Viewer
- Their own profile, coins, and streaks
